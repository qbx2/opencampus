오픈캠퍼스 개발자 사이트
=================================================================================

오픈캠퍼스는 전국 대학교 통합 서비스를 목표로 시작된 프로젝트입니다.
현재 강의 정보 시스템과 기본적인 커뮤니티 기능 또 학교별 확장을 위한 시스템을
지원하고 있습니다.

이 사이트는 오픈캠퍼스 연동이나 오픈캠퍼스 프로젝트에 참가하는 개발자를 위한
문서 페이지입니다.


들어가기 앞서
---------------------------------------------------------------------------------

오픈캠퍼스는 크게 3가지 구성으로 구분됩니다.
아래 구성을 참고하시어 자신에게 필요한 문서를 읽어주세요.

1. App

  오픈캠퍼스와 Restful+OAuth2.0으로 연결되는 외부 서비스를 지칭합니다.
  개발자들은 App을 이용하여 오픈캠퍼스의 다양한 시스템에 접근할 수 있습니다.

2. OpenCampus Gateway Server

  각 학교 시스템들과 연결되는 모듈입니다. Gateway 개발자들은 자신의 학교와
  오픈캠퍼스를 연동하여 오픈캠퍼스에 자신의 학교를 추가할 수 있습니다.

3. OpenCampus Web Server

  여러분이 보고 계시는 이 문서를 포함하여 오픈캠퍼스의 메인 시스템을
  담당하는 서버입니다. AGPL 라이센스로 배포되고 있습니다.
